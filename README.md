# 📅 Система управления графиками

Веб-аналог VBA-макросов для управления производственными календарями, графиками удалённой работы и визуализации отсутствий сотрудников.

## 🚀 Быстрый старт

### Вариант 1: GitHub Pages (рекомендуется)

1. Создайте новый репозиторий на GitHub
2. Загрузите файлы: `index.html`, `style.css`, `app.js`
3. Перейдите в **Settings → Pages**
4. Выберите ветку `main` и нажмите **Save**
5. Через 1-2 минуты сайт будет доступен по адресу: `https://ваш-ник.github.io/имя-репозитория/`

### Вариант 2: Локальный запуск

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в любом браузере
3. Готово!

## 🔧 Как использовать

### Шаг 1: Загрузка производственного календаря

1. Введите год (например, 2026)
2. Выберите способ загрузки:
   - **API** — автоматическая загрузка (базовый календарь)
   - **Excel файл** — загрузка из XLSX
   - **HTML код** — вставка кода страницы consultant.ru
3. Нажмите **«Загрузить календарь»**

### Шаг 2: График удалённой работы

1. Загрузите файл **сотрудников** (XLSX)
   - Лист должен содержать: `ID`, `ФИО`, `Дни удалёнки`
2. Загрузите файл **отсутствий** (XLSX, опционально)
   - Лист должен содержать: `ID`, `Тип`, `Начало`, `Конец`
3. Выберите месяц для формирования графика
4. Нажмите **«Сформировать график удалёнки»**

### Шаг 3: Визуальный календарь

1. Нажмите **«Создать визуальный календарь»**
2. Просмотрите результат в виде таблицы
3. Нажмите **«Скачать все результаты»** для экспорта в Excel

## 📁 Формат входных файлов

### Файл сотрудников (Сотрудники.xlsx)

| ID | ФИО | Дни удалёнки |
|----|-----|--------------|
| 001 | Иванов И.И. | понедельник, среда |
| 002 | Петров П.П. | вторник, четверг |

### Файл отсутствий (Отпуска и больничные.xlsx)

| ID | Тип | Начало | Конец |
|----|-----|--------|-------|
| 001 | Ежегодный отпуск | 01.06.2026 | 14.06.2026 |
| 002 | Больничный удаленно | 10.06.2026 | 15.06.2026 |

**Типы отсутствий:**
- Ежегодный отпуск
- Дополнительный отпуск
- Учебный отпуск
- Больничный
- Больничный удаленно
- Отпуск без сохранения ЗП
- Декретный отпуск
- По уходу за ребенком

## 🔒 Безопасность и приватность

- ✅ **Все данные обрабатываются в браузере** — ничего не отправляется на сервер
- ✅ **LocalStorage** — данные сохраняются только на вашем устройстве
- ✅ **Открытый код** — можно проверить в браузере (F12 → Sources)

## 🛠 Технические требования

- Современный браузер (Chrome, Firefox, Edge, Safari)
- Поддержка JavaScript ES6+
- Подключение к интернету (для загрузки библиотек SheetJS и Tailwind)

## ⚠️ Известные ограничения

1. **CORS** — прямой парсинг consultant.ru из браузера заблокирован. Используйте HTML-вставку или API.
2. **Размер файлов** — рекомендуется до 10 МБ на файл
3. **Количество сотрудников** — оптимально до 500 человек
4. **Мобильные устройства** — интерфейс адаптирован, но таблицы лучше смотреть на ПК

## 🐛 Устранение проблем

| Проблема | Решение |
|----------|---------|
| Календарь не загружается | Попробуйте способ «HTML код» |
| Файл не читается | Проверьте названия колонок (ID, ФИО) |
| Таблица не отображается | Обновите страницу (Ctrl+F5) |
| Данные пропали | Проверьте LocalStorage в браузере |
